
  - Функции
  - это наши действия в JS, которые сокращают код (как циклы)
  - Калькулятор - набор функций (так же холодильник, телевизор)
  - Названия функций - глаголы
  - Анонимные функций - функции без имени, они вызываются и используются здесь и сейчас!
  - При вызове функции ей можно дать данные, которые она будет использовать внутри себя

  -- Аргументы функции - передаются при вызове --
  function showFirstMessage(text) {
    console.log(text);
  }
  
  showFirstMessage("Hello world");
